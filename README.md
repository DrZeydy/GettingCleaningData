# GettingCleaningData
Repository for Getting and Cleaning Data Course Project

This repository contains the following files:
- README.md - this file
- CodeBook.md - describes the data, variables and transformations made to the data
- run_analysis.R - code to transform the data
- tidydata.txt - output file with tidy data set for the project

To produce the tidydata.txt file, source "run_analysis.R" in the directory where the dataset is located (see CodeBook for explanation).  From R prompt, run:
```{r}
run_analysis()
```
